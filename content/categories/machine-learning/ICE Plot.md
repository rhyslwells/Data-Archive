Unlike a Partial Dependence plot, which shows the average effect of the input feature, an ICE plot visualizes the dependence of the prediction on a feature for each sample separately with one line per sample.

https://scikit-learn.org/stable/modules/partial_dependence.html#individual-conditional-expectation-ice-plot
